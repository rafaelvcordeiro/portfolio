Bookmovie


Description
Bookmovie is a lightweight and intuitive Flask/Python webapp to act as a movie bookmarker, aimed to be used whenever need.
Year: 2023
Try it: click to access and use it the way you like it.
Github: project repository
Technologies and tools
Flask: Flask is simple to implement, reliable and light.
SQLite3: This webapp uses a SQLite database with two tables: USERS, that holds users data and RECORDS, that stores data from each user.
Cinemagoer API: to make it fast and easy to add new entries, the Cinemagoer IMDB API was used
Memegen: used the Memegen meme generator for writing automatic error messages and information.
Skeleton: used Skeleton CSS framework to increase frontend development speed.
