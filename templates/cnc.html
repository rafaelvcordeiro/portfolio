CNC Machines


Throughout the years I have been building several CNC machines. I dit it mostly because I really enjoy making automated machines and robots, and also because I wanted to be able to produce high quality parts in a fast and cost-effective way.
CNC stands for Computer Numerical Control, and it is basically an automated machine controlled by a dedicated computer to perform manufacturing operations.
CNC André Ferri
Developed a special CNC machine suited for 3D sculpting of big wooden blocks for the factory of brazilian designer André Ferri
Website: André Ferri Atelier
Year: 2021
Working volume: 1500 x 1000 x 450 mm
Tools: Rhinoceros 3D, SimScale, Solidworks, LinuxCNC
Key Achievements: One of the biggest challenges of this project was to make this machine able to carve hardwood at a 450mm depth travel, and it performed better than expected.
Plywood CNC
Developed a CNC machine suited for cutting an entire plywood sheet (2200x 1600 mm). This machine was developed with the intention of providing precision manufacturing services.
Year: 2015
Working volume: 2200 x 1600 x 140 mm
Tools: Rhinoceros 3D, Solidworks, LinuxCNC
Key Achievements: managed to build an industrial grade CNC machine that is was cost effective and long lasting, being used on a daily basis production and that is still good as a new one.

3D Printer
Developed a different type of CNC machine, a FDM (Fused Deposition Modeling) 3d printer, using an original mechanical layout, to achieve high stiffness and low cost with fewer expensive pieces.
Year: 2014
Working volume: 200 x 200 x 200 mm
Tools: Rhinoceros 3D, Marlin firmware
Key Achievements: well, it works, and actually prints plastic parts using PLA or ABS, until today :D
